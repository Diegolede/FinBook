First off, thank you for considering contributing to Finbook! It's people like you that make the software better for everyone.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tracker.
- Describe the bug and include steps to reproduce.
- Mention the version of the app you are using.

### Suggesting Enhancements
- Open a GitHub Issue to discuss the idea before implementing it.
- Explain why the enhancement would be useful.

### Pull Requests
1. Fork the repository (for contributing purposes only).
2. Create a new branch for your feature or fix.
3. Submit a Pull Request.
4. Ensure your code follows the existing style and matches the monochromatic aesthetic.

## Code of Conduct
Please be respectful and professional in all interactions within this community.

## Restrictions on Clones
As per our [LICENSE](LICENSE), you are not permitted to clone or fork this repository for the purpose of redistributing it as a separate product or service. This repository is for personal use and collaborative improvement only.
